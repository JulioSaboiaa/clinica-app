# Sistema de Atendimento para Clínica Médica

## 📌 Descrição
Sistema web desenvolvido para gerenciamento de consultas médicas, permitindo login de usuários, agendamento de consultas e controle de horários.

---

## 🚀 Tecnologias Utilizadas
- Node.js
- Express
- Vue.js
- Axios
- MongoDB Atlas
- Vercel
- Render

---

## ⚙️ Funcionalidades
- Login de usuário
- Agendamento de consultas
- Definição de data e horário
- Registro de CPF do paciente
- Validação de horários duplicados
- Listagem de consultas
